# 闪印隐私说明

更新日期：2026 年 7 月 22 日

## 本地清单与 iCloud

闪印以本地优先方式保存清单。开启 iCloud 同步后，清单名称和清单项会通过用户自己的 Apple iCloud 账户在设备间同步。关闭 iCloud 同步不影响本地编辑、预览和打印。

## AI 清单生成

AI 生成功能为可选功能，默认不包含开发者提供的额度。用户需要自行选择服务商并填写 Base URL、模型和 API Key。

- API Key 仅保存在当前设备的 Apple Keychain 中，不会写入清单文件、UserDefaults、iCloud、Supabase 或应用日志，也不会发送给闪印开发者。
- 用户根据主题生成新清单时，主题和补充要求会从设备直接发送给所选的第三方 AI 服务商。
- 用户为已有清单补充项目时，主题、当前清单名称和最多前 200 个清单项可能被直接发送给所选的第三方 AI 服务商。
- 只有在用户主动点击“生成”或“重新生成”后，应用才会发送请求。发送目标由用户在 AI 设置中选择的服务商和 Base URL 决定。
- 生成结果会先显示供用户检查和编辑，只有用户确认后才写入本地清单。
- 闪印及其 Supabase 服务不代理 AI 请求，也不保存原始提示词或 AI 响应。第三方 AI 服务商可能根据其自身服务条款和隐私政策处理这些内容。
- AI 配置按设备保存，不会跨设备同步。用户可以随时在设置中删除配置和 API Key。

## 可选产品统计

AI 生成功能与开发者的 Supabase 产品统计相互独立。只有用户另行明确同意“共享清单内容用于产品统计”后，清单名称、清单项文本和随机安装标识才会发送到开发者配置的 Supabase 项目。该数据仅用于产品统计，不用于广告或跨设备同步。用户可以随时关闭共享并删除与当前安装关联的数据。
